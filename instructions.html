Instructions

The game is divided into 3 levels of difficulty

Easy Round:

This round is multiple choice. You have to select the correct answer out of the 4 possible answers within 30 seconds.

Average Round:

This round is problem solving. You have to write the correct answer on the space provided below each question each question within 60 seconds.

Difficult Round:

This is basically the same with average round, but with a higher difficulty but the timer is set to 120 seconds.

Each level will have 5 randomly generated questions and you need to get at least 3 correct answers

(60% of the total score for each level) before you can proceed to the next level.

You will always start in the Easy round. Your scores will be recorded after each attempt.

Have fun.
